### 🌟 Featured Projects

#### 🤖 [Autonomous Exploration Robot & Vision-Based Topological Mapping](https://github.com/alinekoeerad/Self-driving-RC-car)
*B.Sc. Thesis* <br>
An fully autonomous robotic system that explores unknown environments, generates topological maps, and navigates intelligently using **ArUco visual markers** as a "Virtual GPS."
- ✅ **Frontier-based Exploration Algorithm** (enhanced version)
- ✅ **Topological Mapping** with ArUco marker nodes
- ✅ **A* Path Planning** for shortest-path navigation
- ✅ **Python Server** + **C# WPF Controller** architecture
- ✅ Real-time camera feed & live map monitoring

> 🎓 *Submitted as Bachelor of Science thesis at Zanjan University (Sep 2025), supervised by Dr. Sajad Haghzad Klidbary*
